# 💻 Real-Time Collaborative Code Editor

A real-time collaborative coding platform where multiple users can join a room, write code together, chat, and execute code instantly.

---

## 🚀 Features

- 👥 Multi-user real-time collaboration
- 💬 Live chat system
- 🧠 AI code assistant (suggestions & debugging)
- ▶️ Run Python & JavaScript code
- 📡 Socket.io live synchronization
- 🟢 Online / Offline user status
- ⌨️ Typing indicator
- 💾 Code auto-save per room

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- CSS

**Backend**
- Node.js
- Express.js
- Socket.io

**Other**
- Child Process (code execution)
- OpenRouter API (AI integration)

---

## ⚙️ Installation

### 1. Clone repo
```bash
git clone https://github.com/YOUR_USERNAME/collab-code-platform.git
cd collab-code-platform
2. Install dependencies
cd client
npm install

cd ../server
npm install
3. Setup Environment

Create .env in client:

REACT_APP_API_KEY=your_api_key_here
4. Run project
# Start backend
cd server
node server.js

# Start frontend
cd client
npm start
